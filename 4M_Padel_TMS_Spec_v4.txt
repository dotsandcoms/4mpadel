1.  System Overview
The 4M Padel TMS is fully integrated into the 4M Padel website. The federation (SAPA / 4M Padel) sits at the centre — sanctioning events, managing player registrations, and controlling rankings. All data synchronises in real time.

6.  Tournament Formats & Back Draw Options
Format A — Single Elimination with Plates

MAIN DRAW
Knockout bracket — seeded

PLATE / BACK DRAW
Round 1 losers — second chance

Organiser configuration options for Back Draw

	•	Toggle Plate / Back Draw on or off
	•	Set which rounds feed the Plate (default: Round 1 losers; option to include later rounds)
	•	Configure Plate draw size (auto or manual)
	•	Set whether Plate has a 3rd place playoff
	•	Set separate points for Plate positions in the points table
	•	Consolation draw option (for Plate losers) if desired
	•	R16 / R32 playoff bracket options (e.g. 9th place, 10th place, 11th/12th — as per points table columns)


Format B — Group Stage → Upper & Lower Bracket

PHASE 1: Round-Robin Groups — all pairs play each other within their group
▼  Top pairs advance  ▼
UPPER BRACKET
Win = stay  |  Lose = drop to Lower

LOWER BRACKET
Second chance  |  Lose = eliminated
▼  Grand Final  ▼
GRAND FINAL: Upper Bracket winner vs Lower Bracket winner

Organiser configuration options

	•	Number of groups and pairs per group
	•	How many pairs advance per group (e.g. top 2)
	•	Group standings tiebreak: wins → sets → games
	•	Grand Final: single match or best of two
	•	3rd place playoff: on/off
	•	Playoff bracket options matching the points table (9th place, 10th place, 11th/12th, 13th–16th, etc.)

7.  Match Format Configuration
The organiser configures the match format for each category within the tournament. This is set during tournament creation and applies to all matches in that category.

7.1  Sets & Games
Option
Description
Best of 3 sets (full)
Each set played to 6 games. Third set if required. Standard tournament format.
Best of 3 sets with match tiebreak
First two sets to 6. Third set replaced by a match tiebreak to 10 points (win by 2).
2 sets + super tiebreak
Two full sets. If one set each, a super tiebreak to 10 decides. Most common club format.
Single set
One set to 6 games. Used for group stages or time-limited formats.
Pro set
First to 8 games (or configurable number). No second set.

7.2  Scoring Options
Option
Description
Standard scoring
Normal game scoring: 15, 30, 40, deuce, advantage.
No-ad scoring (Golden Point)
At deuce (40-40), one decisive golden point is played. Receiver chooses side. Speeds up matches.
Short sets
Sets played to 4 games (win by 2) rather than 6. Used for Americano or time-restricted events — not in Phase 1.

7.3  Configuring Format per Category
The organiser sets the match format independently for each category within the event. Example:

Category
Sets format
Scoring
Cat 1
Best of 3 sets with match tiebreak
Standard scoring
Cat 2
2 sets + super tiebreak
No-ad (golden point)
Cat 3
2 sets + super tiebreak
No-ad (golden point)
Cat 4
Single set
No-ad (golden point)

	•	The score entry interface adapts automatically to the configured format
	•	Scorecards on the public draw display the format used for each category
	•	The federation admin can set default formats per category as a template for organisers
